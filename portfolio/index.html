Good to know

What's an "AI Engineer" (vs an ML Engineer)?+
An ML Engineer trains models from scratch (heavy math). An AI Engineer builds real products on top of existing models like GPT and Claude — using RAG, agents and vector databases. I'm the second kind: I make AI useful inside apps.
What does "on-device AI" mean?+
The AI model runs directly on the phone instead of a server — so it works offline, is fast, and keeps data private. I use TensorFlow Lite / LiteRT and ML Kit for this.
What is RAG?+
Retrieval-Augmented Generation. It lets an AI answer using your documents: it finds the relevant pieces first, then writes an answer grounded in them — with far fewer made-up facts.
Can you build the whole thing, front to back?+
Yes — the Android app, the backend API, and the AI layer. That's the point of this portfolio: one person, the full stack of an AI mobile feature.
